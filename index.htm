Regular Expression Editor

Install

  • Unzip the org.microprog.regex_1.0.0_plugin.zip file to $ECLIPSE_HOME / plugins

This plugin is for the interface for jakarta-oro package for regular expression. It supports for the Perl, Awk and Glob type regular expressions.

Working in it:

1. Select Window -> Customize perspective -> File -> New -> RegEx Editor File

2. Select File -> New

3. Click the RegEx Editor File and it will open a wizard, give a file name with extension ".rx".

4. Next it will open the editor.

4. To see the Result , click Expression Results, which is placed in the bottom of the editor.
